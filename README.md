# 🌿 Galia — Chatbot WhatsApp de Galiebra Beauty

## Variables de entorno necesarias en Render.com

| Variable | Dónde obtenerla |
|---|---|
| GEMINI_API_KEY | Google AI Studio → aistudio.google.com/apikey |
| WHATSAPP_TOKEN | Meta for Developers → Configuración de la API |
| PHONE_NUMBER_ID | Meta for Developers → Identificador del número |
| VERIFY_TOKEN | Lo inventas tú (ej: galiebra2024) |

## URL del Webhook para Meta
Una vez desplegado en Render, tu URL será:
https://TU-APP.onrender.com/webhook
